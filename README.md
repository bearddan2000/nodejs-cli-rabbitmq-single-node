# nodejs-cli-rabbitmq-single-node

## Description
A single node producer/consumer
message broker demo.

## Tech stack
- nodejs

## Docker stack
- node:latest
- rabbitmq:3.6.6-management

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
